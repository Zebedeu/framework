# Installation

Create project

composer create-project --prefer-dist knut7/knut7 site

## Install

composer.json file:
```json
{
    "require": {
        "knut7/framework": "*"
    }
}
```
after run the install command.
```
$ composer install
```

OR run the following command directly.

```
$ composer require knut7/framework
```
